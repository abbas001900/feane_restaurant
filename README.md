# Feane Restaurant

Vitrine responsive pour un restaurant fictif nommé **Feane Restaurant**.

## Description
Site statique en HTML et Tailwind CSS, conçu pour présenter un restaurant avec :
- une page d'accueil présentant un hero full-width, un slider textuel et des sections d'information
- un menu de navigation responsive avec affichage desktop et menu mobile basculant
- des pages internes pour `about`, `menu` et `book_table`
- un style visuel centré sur le contraste sombre, les accents orange et les images de restauration

## Fonctionnalités
- navigation responsive avec un toggle hamburger sur mobiles
- menu mobile animé en CSS via `checkbox` et classes Tailwind
- sections de présentation et de menu produit
- boutons call-to-action stylés et icônes Font Awesome

## Structure
- `index.html` : page d'accueil
- `src/pages/about.html` : page « about »
- `src/pages/menu.html` : page menu
- `src/pages/book_table.html` : page de réservation
- `images/` : visuels et photos utilisés
- `src/output.css` : feuille de style finale / Tailwind généré

## Technologies
- HTML
- Tailwind CSS
- Font Awesome
- CSS responsive
